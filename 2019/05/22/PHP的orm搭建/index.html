定义findOne方法，使用方法为:

include(BasicModel.php);

单个查询条件时:

定义findAll方法，使用方法同findOne一样。

定义findlimitAll方法，使用方法为:

include(BasicModel.php);

后面一个参数为分页的页码数

$label = findlimitAll([‘id’=>1],3);

inclue(‘BasicModel.php’);
