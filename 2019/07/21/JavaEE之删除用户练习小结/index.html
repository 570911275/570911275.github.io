练习

删除用户练习

  • 思路：

    1. 点击删除按钮（携带该用户的id）
    2. 跳转确认画面，让客户确认是否删除
    3. 确认删除后
    4. 删除数据
    5. 跳转回原界面

  • 文件跳转：

    1. jsp获取用户选择删除用户id
    2. 跳转至Servlet层，执行业务逻辑
    3. 在Servlet中，将使用到Service
    4. 在Service中，将使用到Dao
